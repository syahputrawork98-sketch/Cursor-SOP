# SR-04: AI Updates & Living Knowledge

Sub-Rak ini berfungsi untuk menjaga seluruh perpustakaan **Cursor-SOP** tetap relevan dengan perkembangan dunia AI yang sangat cepat. Di sini kita mencatat update model, fitur baru IDE, dan revisi best practice.

---

## 📚 Buku Utama

- [BK-01: Model Updates](./BK-01-Model-Updates.md) — Saat ada rilis model baru (GPT-x, Gemini x.x, dll).
- [BK-02: Feature Updates](./BK-02-Feature-Updates.md) — Update fitur Cursor atau IDE pendukung lainnya.
- [BK-03: Best Practice Revisions](./BK-03-Best-Practice-Revisions.md) — Saat kita menemukan cara kerja yang lebih efektif.

---

## 🔄 Protokol Sinkronisasi

Jika ada update besar di buku-buku di atas, pastikan untuk:
1. Update `CHANGELOG.md` di root.
2. Update RAK yang relevan (misal: fitur baru Cursor diupdate ke RAK-05).
3. Informasikan kepada AI Agent di sesi berikutnya agar membaca update terbaru ini.
