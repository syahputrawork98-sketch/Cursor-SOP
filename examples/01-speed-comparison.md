# Laboratory 01: Speed Comparison

## Objective
Membandingkan kecepatan antara penulisan manual, autocomplete, dan agentic generation.

## Steps
1. Buka file kosong `test.js`.
2. Coba tulis fungsi `fetchUserData` secara manual. Catat waktunya.
3. Gunakan Tab-Autocomplete Cursor untuk fungsi yang sama.
4. Gunakan `Ctrl+K` (Compose) untuk menulis seluruh modul.

## Results
Anda akan melihat bahwa Agentic Generation (Ctrl+K) tidak hanya lebih cepat, tapi juga menyertakan error handling yang sering terlupa pada penulisan manual.
