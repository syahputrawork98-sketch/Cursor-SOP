# Laboratory 04: Check Index Status

## Objective
Memverifikasi apakah repositori Anda sudah terindeks dengan benar oleh Rust Indexer.

## Steps
1. Buka Cursor Settings (`Ctrl+Shift+J`).
2. Masuk ke tab **General** -> **Features**.
3. Lihat bagian **Codebase Indexing**.
4. Pastikan statusnya **Indexed**.
5. Coba cari file yang baru saja dibuat menggunakan `@codebase`.

## Troubleshooting
Jika status "Paused", klik "Resync Index" untuk memaksa pemindaian ulang.
