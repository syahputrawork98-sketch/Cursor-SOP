# RAK-08: Sinkronisasi Global (Bahasa Manusia)

## 🌟 Gampangnya...
Rak ini adalah tentang **"Kerapihan Lintas Proyek"**. Memastikan standar koding Anda di Python sama rapinya dengan di JavaScript.

## 🥁 Ritme Harian (The Rhythm of Work)
Gunakan ritme ini saat Anda ingin melakukan pembersihan besar-besaran (Spring Cleaning):
1.  **Ritme Normalisasi**: Samakan gaya bahasa.
    - *Ketik:* *"Audit seluruh folder ini, pastikan gaya penamaan variabelnya sudah konsisten."*
2.  **Ritme Sapu Bersih**: Hapus yang tidak berguna.
    - *Ketik:* *"Cari file-file sampah atau kode yang sudah tidak dipakai lagi, buatkan daftar untuk dihapus."*
3.  **Cross-Project Sync**: *"Bandingkan struktur folder ini dengan proyek saya yang ada di @RAK-lain, pastikan polanya sama."*

## 📋 Ceklis Mandor (The Checklist)
- [ ] Apakah semua file sudah mengikuti gaya penamaan yang sama?
- [ ] Apakah proyek sudah terlihat seperti hasil kerja satu tim (bukan acak-acakan)?
- [ ] Apakah Anda sudah melakukan "Audit Terakhir" sebelum menutup hari?
