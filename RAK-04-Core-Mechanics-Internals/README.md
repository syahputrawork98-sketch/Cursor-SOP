# RAK-04: Cara Berpikir Cursor (Bahasa Manusia)

## 🌟 Gampangnya...
Rak ini adalah tentang **"Daya Ingat AI"**. Cursor punya batas ingatan, dan tugas Anda adalah menjaganya agar tetap fokus.

## 🥁 Ritme Harian (The Rhythm of Work)
Gunakan ritme ini agar AI tidak "pikun" atau "halusinasi":
1.  **Bersihkan Konteks**: Jika topik pembicaraan berubah, bersihkan chat atau buang file yang tidak relevan.
    - *Tindakan:* Klik tanda silang (x) pada file di atas kolom chat yang sudah tidak dipakai.
2.  **Panggil Referensi Spesifik**: Jangan biarkan AI mencari sendiri di ribuan file. 
    - *Ketik:* *"Gunakan file @nama_file sebagai referensi utama, abaikan yang lain."*
3.  **Kontrol Index**: Jika AI bilang tidak tahu, suruh dia rescan: *"Coba lakukan Indexing ulang pada folder ini."*

## 📋 Ceklis Mandor (The Checklist)
- [ ] Apakah AI membaca file yang benar?
- [ ] Apakah chat sudah terlalu panjang? (Kalau ya, buat chat baru agar AI segar lagi).
- [ ] Apakah simbol `@` sudah digunakan dengan tepat?
