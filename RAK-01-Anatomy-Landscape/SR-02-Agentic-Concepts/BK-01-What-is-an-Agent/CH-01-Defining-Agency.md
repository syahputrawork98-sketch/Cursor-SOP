# CH-01: Defining Agency

## 📖 1. The Core Definition
Apa yang membedakan **AI Agent** dengan **LLM (Large Language Model)** biasa? LLM adalah mesin prediksi teks; AI Agent adalah LLM yang diberikan **Tujuan (Goal)** dan **Kekuasaan (Agency)** untuk melakukan aksi demi mencapai tujuan tersebut.

## ⚙️ 2. Capabilities of an Agent
- **Memory**: Kemampuan menyimpan pengalaman masa lalu (short-term & long-term).
- **Tool Use**: Kemampuan berinteraksi dengan dunia luar (browser, terminal, API).
- **Planning**: Kemampuan memecah tugas besar menjadi langkah-langkah kecil.

## 📊 3. Comparison Table
| Feature | Basic LLM | AI Agent |
| :--- | :--- | :--- |
| **Input** | User Prompt | Objective / Goal |
| **Output** | Text Response | Executed Action / Changes |
| **Logic** | One-shot | Circular / Interactive |
| **Access** | Sandbox | File System / Tools |

## ⚠️ 4. The Illusion of Agency
Hati-hati: Seringkali AI hanya "berakting" seolah-olah dia agen padahal ia hanya mengikuti instruksi teks tanpa benar-benar memverifikasi hasil aksinya. Agentic AI yang sesungguhnya harus memiliki **Feedback Loop**.
